## Requirements
- poetry

## Installing

- Create virtual environment
```
virtualenv .venv
```

- Activate virtual environment
```
source .venv/bin/activate
```

- Install dependencies
```
poetry install
```

## Getting started

- Start bot
```
poetry run python main.py
```
